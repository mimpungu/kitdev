
Deo - Web Optimizer
--------------------

Ce kit permet d'optimiser facilement vos fichiers HTML, CSS, JS, PHP, Java, Dart et Flutter.

Fonctionnalités :
- Minification HTML/CSS
- Obfuscation JavaScript
- Compilation Java/Dart
- Copie directe de fichiers PHP
- Interface graphique (Python) utilisable sur tous systèmes

Fichiers inclus :
------------------
- script-optimize.sh        → Script shell pour Linux/macOS
- script-optimize.bat       → Script batch pour Windows
- script-personnalise.sh    → Script exemple modifiable
- interface_optimiseur.py   → Interface graphique Python
- README.txt                → Ce fichier

Pré-requis :
------------
- Python 3 avec Tkinter
- Outils CLI recommandés : html-minifier, cleancss, javascript-obfuscator, minify
  (Utilise automatiquement celui qui est disponible)

Utilisation :
-------------
- Double-cliquez sur interface_optimiseur.py ou lancez-le avec python3
- Ou utilisez les scripts .sh ou .bat selon votre OS
